Let's design an iPhone/iPad app called "Dictee" for learning French words for dictation.
Input - is a list of new French words to learn, given by a teacher to a pupil (user of the app) as a list of paper.
App should allow user to make a photo of this list of paper, and then recognizes words in it.

Then app starts a teaching session - it randomly chooses a word from this list, and the pupil must type the correct spelling.
App goes through all the words in the list, and only in the end grades a pupil - shows which words he/she wrote correctly, and which incorrectly.
User can repeat such learning session any times in the future per each photographed list of words.

App remembers incorrectly typed words.
Then user can start a special session called Revisit, where app will show only words that user typed incorrectly. If in this session user types any of the words correctly, app removes them from the remembered list of words to review.
